我将为您开发一个 Node.js CLI 工具，用于通过 Git 地址和分支名快速创建 Worktree。

## 项目结构
- **语言**: TypeScript (提供类型安全和更好的维护性)
- **运行环境**: Node.js
- **核心库**:
  - `commander`: 用于解析命令行参数
  - `simple-git`: 用于执行 Git 操作
  - `chalk`: 用于美化终端输出

## 实现步骤
1.  **项目初始化**:
    -   创建 `package.json` 和 `tsconfig.json`。
    -   安装必要依赖 (`commander`, `simple-git`, `chalk`, `typescript`, `ts-node`)。

2.  **开发 CLI 核心逻辑 (`src/index.ts`)**:
    -   **输入参数**: 接收 `Git仓库地址` (repoUrl) 和 `分支名称` (branchName)。
    -   **仓库管理**:
        -   从 URL 解析仓库名称。
        -   检查本地是否存在该仓库的 "bare" (裸) 仓库。
        -   如果不存在，自动执行 `git clone --bare <url>` 进行克隆。
    -   **Worktree 创建**:
        -   创建目录结构，例如 `<仓库名>/<分支名>`。
        -   执行 `git worktree add` 命令，为指定分支创建独立的工作目录。

3.  **配置与构建**:
    -   在 `package.json` 中配置 `bin` 字段，使工具可执行。
    -   添加编译和运行脚本。

4.  **验证**:
    -   使用一个测试仓库验证工具能否正确下载代码并创建 Worktree。
